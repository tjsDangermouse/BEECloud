Understanding the Home screen

Temperature ring

The circle in the middle tells you the current room comfort and target temperature. Drag the ring or tap the arrows to nudge the target.

Quick buttons

Boost hot water or enable Holiday mode with a single tap. The buttons fade slightly when the action is already active.

Next events

The top-left corner highlights the next heating and hot-water schedule. Handy if you want to see what is coming up.

Keep the system healthy

  • Status dots: tap any coloured dot to read the plain-language explanation. Red entries come with suggested fixes and links.
  • Auto-fetch: the dashboard collects new readings in the background. Turn it on or off in Settings if you need to pause requests.
  • Logs: peeking inside melcloud.log or the Monitor health script can help if something unusual keeps returning.

Weekly habit

Spend 60 seconds on the Energy page each week. Sudden colour changes highlight when doors were left open or schedules need tweaking.

Technical notes (optional)

Useful API endpoints

/api/data returns the latest readings, /api/health summarises the service, and /api/auto-fetch/toggle lets scripts pause collection. All endpoints respond with JSON.

Database care

Back up instance/melcloud.db regularly. If it becomes read-only, POST to /api/database/recover or run the recovery action from the monitoring script.

Docker deployment

docker-compose up -d starts the stack, docker-compose pull fetches updates, and the container reports its health status via /api/health.

Technical toolkit

API highlights

/api/data, /api/status, and /api/health feed dashboards, home automations, or reporting scripts.

Auto-fetch control

Use /api/auto-fetch/status to confirm collection, /api/auto-fetch/toggle to pause, and /api/auto-fetch/trigger to force a run.

Database safety

Keep regular backups of instance/melcloud.db. Restore access with /api/database/recover if the database becomes read-only.

Docker commands

docker-compose up -d --build recreates the containers with local changes, and docker logs melcloud -f streams live feedback.

Frequently asked questions

How often is data refreshed?

Auto-fetch runs on the interval you choose in Settings (default every few minutes). Manual refresh is always available from the Home screen.

Can I invite others?

Yes. Settings ‚Üí Users lets you add or remove accounts with tailored permissions.
